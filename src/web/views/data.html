인플루언서 데이터는 스크래핑으로 수집된 모든 프로필의 마스터 데이터입니다. AI 분류를 통해 인플루언서/업체를 자동 판별하고 국가를 감지합니다. AI 분류 실행 후 캠페인에 자동 배정됩니다.

키워드 타겟은 자동 스크래핑의 핵심입니다. 플랫폼과 국가를 선택하고 키워드를 입력하면 페어 ID가 자동 생성됩니다. 주기에 설정한 시간(기본 72시간)마다 자동으로 스크래핑이 실행되며, 1회 실행에는 포스트 수집 + 프로필 분석이 모두 포함됩니다. Until을 설정하면 해당 날짜 이후의 게시물은 수집하지 않습니다. Run 버튼을 누르면 즉시 스크래핑이 시작됩니다.

DM 캠페인으로 수집된 인플루언서에게 자동 DM을 발송합니다. 캠페인 생성 시 발송 계정 쿠키를 직접 등록하세요. 캠페인 생성 → 큐 생성(타겟 배정) → 시작 순서로 진행합니다. 스크래핑으로 수집된 프로필은 국가 + 플랫폼 매칭으로 자동 배정됩니다.

댓글 템플릿은 DM 발송 전 인플루언서 게시물에 자동 댓글을 작성할 때 사용됩니다. 봇 감지를 방지하기 위해 다양한 템플릿(5개 이상 권장)을 등록하세요.
