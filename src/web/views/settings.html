Platform Cookie Status

To enable scraping, export your browser cookies for each platform using a browser extension (e.g., EditThisCookie) and save them as JSON files in the cookies/ directory.

How to Add Cookies

  1. Install a cookie export browser extension (e.g., EditThisCookie for Chrome)
  2. Log in to the target platform (Instagram, Twitter, etc.)
  3. Click the extension and export cookies as JSON
  4. Save the JSON file as cookies/<platform>.json
  5. Refresh this page to verify the cookies are loaded
