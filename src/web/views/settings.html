설정

설정에서는 스크래핑용 플랫폼 쿠키와 캠페인별 DM 발송 쿠키를 확인합니다. 스크래핑을 시작하려면 먼저 해당 플랫폼의 쿠키를 등록해야 합니다. EditThisCookie 등의 확장 프로그램으로 쿠키를 추출하여 저장하세요.

플랫폼 쿠키 상태 (스크래핑용)

브라우저 확장 프로그램으로 각 플랫폼의 쿠키를 추출하여 cookies/ 디렉토리에 JSON 파일로 저장하세요.

캠페인 DM 쿠키 (캠페인별 발송 계정)

각 캠페인에 등록된 발송 계정 쿠키 상태입니다. 캠페인 관리에서 쿠키를 업로드하세요.

등록된 캠페인이 없습니다. 캠페인 생성을 먼저 진행하세요.

쿠키 추가 방법

  1. 쿠키 추출 브라우저 확장 프로그램을 설치하세요 (예: Chrome용 EditThisCookie, Firefox용 Cookie-Editor)
  2. 대상 플랫폼(Instagram, Twitter, TikTok)에 로그인하세요
  3. 확장 프로그램에서 쿠키를 JSON으로 내보내세요
  4. 스크래핑용: cookies/<플랫폼>.json 으로 저장
  5. DM 캠페인용: 캠페인 관리에서 Cookie 버튼으로 직접 업로드
  6. 이 페이지를 새로고침하여 쿠키가 로드되었는지 확인하세요

플랫폼별 필수 쿠키:

Instagram: sessionid, csrftoken, ds_user_id

Twitter: auth_token, ct0, twid

TikTok: sessionid, tt_csrf_token, msToken, ttwid
